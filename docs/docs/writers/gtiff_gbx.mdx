# Lightweight GeoTIFF Writer (`gtiff_gbx`)

`gtiff_gbx` is the [`raster_gbx`](./raster_gbx) catch-all writer with the GeoTIFF
driver forced — output is always GeoTIFF regardless of `tile.metadata`. Register
the lightweight DataSources first (see
[Lightweight Raster Writer](./raster_gbx)).
