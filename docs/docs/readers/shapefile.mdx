Read ESRI Shapefile format using the `shapefile` reader.

## Format Name

`shapefile_ogr`

## Supported Files

- **`.shp`** - Standard shapefile (requires .shx, .dbf files)
- **`.zip`** - ZIP files containing shapefiles
- Directories with multiple shapefiles
