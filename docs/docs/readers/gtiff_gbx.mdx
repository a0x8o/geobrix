# Lightweight GeoTIFF Reader (`gtiff_gbx`)

`gtiff_gbx` is the [`raster_gbx`](./raster_gbx) catch-all reader with the GeoTIFF
driver preset — use it to make GeoTIFF reads explicit. It is pure-Python (no JAR)
and emits the same `(source, tile)` schema as the other readers.

Register the lightweight DataSources first (see
[Lightweight Raster Readers → Register](./raster_gbx#register)).
